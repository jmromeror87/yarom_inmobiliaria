# YaRom Inmobiliaria ERP

**YaRom Inmobiliaria ERP** es un **ERP inmobiliario inteligente** diseñado para la gestión integral de propiedades en arriendo.
El sistema automatiza todo el ciclo del negocio inmobiliario:

**Solicitud → Contrato → Facturación → Contabilidad → Pago a propietarios**

Incluye **contabilidad bajo NIIF**, automatización de procesos y módulos de **inteligencia artificial para análisis de riesgo y rentabilidad**.

---

# Objetivo del Proyecto

Crear una plataforma tecnológica que permita a administradoras inmobiliarias:

* Automatizar la gestión de arrendamientos
* Controlar contabilidad bajo normas NIIF
* Reducir riesgo de mora
* Analizar rentabilidad de propiedades
* Centralizar contratos y documentos
* Integrarse con notarías para autenticación contractual
